CSS 2 – Espaçamentos e Estilos Visuais

1. Padding

Você vai usar o padding para criar espaço interno que afasta o conteúdo da borda. Vamos explorar as propriedades padding-top, padding (shorthand) e variar unidades. 

.caixa {
  padding: 16px;        /* todos os lados */
  padding: 8px 16px;    /* vertical 8px, horizontal 16px */
}

2. Border

Iremos aplicar border para desenhar linhas ao redor do padding. Estudaremos as propriedades border-width, border-style, border-color e border-radius. 

.caixa {
  border: 2px solid #333;   /* largura, estilo e cor */
  border-radius: 8px;       /* cantos arredondados */
}

3. Margin

Vamos usar margin para criar espaço externo que separa elementos vizinhos. Iremos experimentar a shorthand margin e margens individuais (margin-top, etc.). 

.caixa {
  margin: 24px auto;    /* vertical 24px, horizontal centralizado */
}

4. Cores e Variáveis CSS

Iremos definir cores em diferentes formatos (keyword, hex, RGB, HSL) e criar variáveis CSS em :root, usando-as com var(--nome). 

:root {
  --cor-primaria: #3498db;
}
h1 {
  color: var(--cor-primaria);
}

5. Backgrounds

Você vai explorar as propriedades de fundo: background-color, background-image, background-size, background-position e background-repeat. 

body {
  background-color: #f0f0f0;
  background-image: url('fundo.jpg');
  background-size: cover;
  background-repeat: no-repeat;
  background-position: center;}

Na aula, você praticará cada um desses conceitos criando caixas, cartões e fundos para entender o impacto de cada propriedade no layout.
