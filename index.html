CSS 1: Introdução

Na aula de hoje aprendemos o que é CSS e como aplicá-lo: vimos as três formas de incorporação (inline, interno e externo), entendemos a estrutura de uma regra CSS (seletor { propriedade: valor; }), trabalhamos seletores básicos (elemento, classe, ID e agrupados), exploramos o modelo de caixa (conteúdo, padding, border e margin), usamos propriedades essenciais como color, background-color, font-family, font-size, margin e padding, diferenciamos elementos block e inline e conhecemos categorias de valores (keywords, unidades e cores em hex/RGB).
