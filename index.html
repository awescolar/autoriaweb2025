CSS 1: Introdução

Na aula de hoje aprendemos o que é CSS e como aplicá-lo: vimos as três formas de incorporação (incorporado - na própria página), entendemos a estrutura de uma regra CSS (seletor { propriedade: valor; }), trabalhamos seletores básicos (elemento e elementos aninhados), exploramos o modelo de caixa (conteúdo, padding, border e margin), usamos propriedades essenciais como color, background-color, border-color, font-size, margin e padding, diferenciamos elementos block e inline e conhecemos categorias de valores (keywords, unidades e cores em hex/RGB).
