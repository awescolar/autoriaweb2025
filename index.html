Resumo Prático de CSS

Este guia reúne, de forma concisa e prática, os principais conceitos e propriedades de CSS vistos em aula. Use-o como referência rápida ao estilizar suas páginas.

1. Sintaxe Básica

seletor {
  propriedade: valor;
}
  • Seletor: elemento HTML, classe (.classe), ID (#id) ou agrupamento (h1, p).
  • Declaração: bloco entre chaves.
  • Propriedade: nome da característica (ex.: color).
  • Valor: valor atribuído (ex.: red).

2. Seletores

  • Elemento: p { … }
  • Classe: .menu { … }
  • ID: #header { … }
  • Agrupado: h1, h2, h3 { … }
  • Descendente: nav ul { … }
  • Filho direto: ul > li { … }

3. Cores

Formatos de cor:

  • Palavra-chave: red, blue
  • Hexadecimal: #RRGGBB (ex.: #ff0000)
  • RGB: rgb(255, 0, 0)
  • HSL: hsl(0, 100%, 50%)

Variáveis CSS:

:root {
  --cor-primaria: #3498db;
}
h1 {
  color: var(--cor-primaria);
}

4. Fundo (Background)

body {
  background-color: #f0f0f0;
  background-image: url('fundo.jpg');
  background-repeat: no-repeat;
  background-size: cover;
  background-position: center;
}

5. Bordas (Border)

.box {
  border: 2px solid #333;
  border-radius: 8px;
}
  • Shorthand: border: 2px solid #333;
  • Individual: border-top, border-right, etc.

6. Espaçamentos: Padding & Margin

Padding (interno):

.container {
  padding: 20px;           /* todos os lados */
  padding: 10px 20px;      /* vertical 10px, horizontal 20px */
  padding: 5px 10px 15px;  /* top, hor., bottom */
}

Margin (externo):

.container {
  margin: 30px; 
  margin: 10px 0;          /* vertical, horizontal */
}

8. Display

div { display: block; }
span { display: inline; }
button { display: inline-block; }
  • block: ocupa toda largura, quebra linha.
  • inline: ocupa só conteúdo, não quebra.
  • inline-block: mistura ambos.

9. Valores e Unidades

  • Absolutas: px, pt.
  • Relativas: %, em, rem, vh.
  • Keywords: auto, inherit.

Dica Final

Organize seus estilos em styles.css e vincule com:

<link rel="stylesheet" href="styles.css">
