CSS 1: Introdução

1. Sintaxe Básica

Em CSS, cada regra (rule set) segue este formato fundamental: 

seletor { propriedade: valor; }
  • Seletor: indica quais elementos recebem o estilo (por ex. h1, .classe, #id).
  • Declaração: o bloco entre chaves { … } que agrupa propriedades.
  • Propriedade: o nome da característica que queremos ajustar (por ex. color).
  • Valor: o valor atribuído à propriedade (por ex. red, 16px).

Exemplo prático:
Definindo cor vermelha e fonte de 24px para todos os títulos h1: 

h1 {
  color: red;
  font-size: 45px;
  margin: 5px 0 20px 5px;
}

2. Seletores Agrupados

Seletores agrupados permitem aplicar a mesma regra de estilo a vários elementos de uma vez. Basta listar cada seletor separado por vírgula.

Use vírgulas para aplicar um mesmo estilo a vários elementos:

  • h1, h2 e p recebem a mesma cor e tamanho de fonte.
  • Você pode agrupar .classe1, .classe2 ou qualquer combinação de seletores.
  • Use agrupamento para reduzir código repetido e manter seu CSS mais limpo.

Experimente agrupar diferentes seletores e observar como todos eles são afetados pela mesma regra. 

h1, h2, p {
  color: teal;
  font-family: Arial, sans-serif;
}

3. Box Model

Todo elemento HTML é uma “caixa” composta por quatro camadas: 

┌───────────────────────┐
│       margin          │ ← Espaço externo entre elementos
│ ┌───────────────────┐ │
│ │      border       │ ← Borda ao redor da caixa
│ │ ┌───────────────┐ │ │
│ │ │   padding     │ │ │ ← Espaço interno entre conteúdo e borda
│ │ │ ┌───────────┐ │ │ │
│ │ │ │ content   │ │ │ │ ← Texto, imagem, etc.
│ │ │ └───────────┘ │ │ │
│ │ └───────────────┘ │ │
│ └───────────────────┘ │
└───────────────────────┘
  • Content: área onde fica o texto ou imagem.
  • Padding: afasta o conteúdo da borda.
  • Border: linha ao redor do padding.
  • Margin: afasta a borda de elementos vizinhos.

Exemplo de uso:


.caixa {
  /* Padding interno */
  padding: 16px;
  /* Borda ao redor */
  border: 2px solid #333;
  /* Margem externa */
  margin: 24px;
  /* Cor de fundo para visualizar */
  background-color: #eef;
}

Experimente alterar padding, border e margin para ver como a caixa se comporta.

Exemplo:


                        
p {
  background-color: #eef;
  padding: 20px;
  border: 2px solid #333;
  margin: 30px;
}

4. Block vs Inline

Block: ocupa toda a largura e quebra linha antes e depois
Inline: ocupa só o espaço do conteúdo e não quebra linha 

                            
p { display: block; background: #def; }
a { display: inline; color: blue; }

5. Categorias de Valores

Em CSS, cada “propriedade” (como cor, tamanho ou margem) precisa de um “valor”. Esses valores se enquadram em algumas categorias simples:

  • Palavra-chave (Keyword): nomes pré-definidos, como red, bold ou auto.
    Exemplo: display: block;
  • Comprimento (Length): números acompanhados de unidade, como px, em ou %.
    Exemplo: margin: 20px;
  • Cor (Color): nome (blue), hexadecimal (#00f), RGB (rgb(0,0,255)) ou HSL (hsl(240,100%,50%)).
    Exemplo: background-color: #ffc107;
  • String: texto entre aspas, usado em propriedades como font-family.
    Exemplo: font-family: "Arial", sans-serif;
  • Função: valores produzidos por funções que calculam cores, tamanhos etc.
    Exemplo: width: calc(100% - 20px);

Pense assim: cada propriedade espera um tipo certo de “resposta” — seja um nome, um número com medida, uma cor, um texto ou o resultado de uma função.

Exemplo:


h1 {
  color: #333;
  background-color: rgb(240,240,240);
  font-family: "Courier New";
  width: 80%;
}
