1. Padding

Você deverá usar Padding nas laterais do card. 

.card-img {
  padding-left: 24px;
  padding-right: 24px;
}

2. Margin

Será necessário aplicar marging entre elementos. 

.card-img {
  margin-top: 24px;
  margin-bottom: 24px;
}
